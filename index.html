latest project

Jukify.de is an appliaction to provide jukebox-functionality in a fair and easy way. Using the spotify API the user can create his own playlist and invite his friends to join a session, each providing their own playlist.

The playlists are combined and played back in a random order. Jukify is a project that started during university classes, but is still receiving updates occasionally. It was my responsibility to create the frontend for the app.
